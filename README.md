## 运行本项目

`docs` 中是基于 `vitepress` 构建的个人博客。

先安装 `pnpm`
```sh
npm i -g pnpm
# 安装依赖
pnpm install
```

运行
```sh
# 运行博客
pnpm dev
```

构建
```sh
pnpm build
```
